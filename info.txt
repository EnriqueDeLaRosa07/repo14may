Para alojar ( o campartir) un repositorio Git en GitHub, siga los siguientes pasos:

Paso 1: Registra una cuenta en Github
El primer paso para comenzar con el aloamiento en Github es crear una cuenta personal. Visite la página de registro oficial para registrarse.

Paso 2: Crear un repositorio remoto en Github
Después de registrarse, crear un home (un repositorio) en Github para el repositorio Git que quieres compartir.
